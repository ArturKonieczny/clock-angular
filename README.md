# clock-angular
Clock written in JS with Angular.

## Sample
https://arturkonieczny.github.io/clock-angular/

## Usage and Development
Download this repository and

```bash
npm install
```

run

```bash
ng serve
```

to start testing enviroment.

Other scripts:

```bash
ng build --prod
ng test
ng lint
ng e2e
```
